== Installation ==

1. Upload the plugin files to `/wp-content/plugins/accesstive`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Access the Accesstive dashboard from the admin menu
4. The assistant widget will automatically appear on your site

== Frequently Asked Questions ==

= Do I need coding knowledge to use Accesstive? =

No, Accesstive is designed for users of all technical levels. Simply install and activate the plugin.

= Will this slow down my website? =

No, the assistant script loads asynchronously and is optimized for performance.

= Is this WCAG compliant? =

Accesstive helps address many accessibility barriers and supports WCAG guidelines, though complete compliance may require additional measures.
